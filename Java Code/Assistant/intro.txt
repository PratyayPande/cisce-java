Binary Tree (0.0.1)

An user friendly command line interface for the implementation of a binary tree to store integer values.
We currently have functions to insert nodes and display the tree.
If you want to add more functions to the existing code, you are always welcome.
Just look up the github page for more information:

    https://github.com/PratyayPande/cisce-java

If you are starting this the first time, then please start by typing in:
    starting-point

Please always insert a value in the root node to initialize the tree by typing:
    insert root <YOUR INTEGER VALUE>
If you need help with a function please type in:
    help <FUNCTION NAME>
Example: help insert
